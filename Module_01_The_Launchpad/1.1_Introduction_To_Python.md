# 1.1 Introduction to Python

## Actually, Why Python?

Imagine you're sitting at your desk.

You have a calculator.

A notebook.

A bottle of water.

And 10,000 students whose average marks need to be calculated.

Suddenly, your calculator starts sweating.

You start sweating.

The notebook starts looking nervous.

At this point, you have two options:

### Option A

Calculate everything yourself.

This is commonly known as:

> "A terrible idea."

### Option B

Convince a computer to do the work for you.

Computers are ridiculously fast.

The problem is that they're also ridiculously literal.

If you tell a human:

> "Please arrange these numbers."

They'll probably figure out what you mean.

If you tell a computer:

> "Arrange these numbers."

The computer responds:

> "Arrange them HOW?"

Ascending?

Descending?

Alphabetically?

By emotional significance?

Computers require extremely precise instructions.

And that's where programming languages come in.

---

## So What Is Python?

Python is a programming language.

In simple terms:

> Python is a language humans use to boss computers around.

Politely.

Most of the time.

Instead of speaking in complicated machine instructions made of 1s and 0s, we write instructions in Python and let Python handle the translation.

For example:

```python
print("Hello, World!")
```

Even if you've never programmed before, you can probably guess what this does.

That's one of Python's greatest strengths:

**It often looks almost like English.**

---

## Why Not Just Use Machine Code?

Technically, we could.

We could also travel everywhere by crawling.

Neither is particularly efficient.

Early programmers had to work much closer to the machine.

Modern programming languages exist because humans got tired of suffering.

Python allows us to focus on solving problems instead of wrestling with hardware details.

---

## Why Is Python So Popular?

Python has become one of the world's most widely used programming languages because it is:

### Easy to Read

Compare:

```python
print("Hello")
```

to mysterious lines of code that look like somebody dropped a keyboard down a staircase.

Python values readability.

### Beginner-Friendly

Python lets beginners build useful programs quickly.

You spend less time fighting syntax and more time learning ideas.

### Powerful

Python is used for:

* Artificial Intelligence
* Robotics
* Data Science
* Scientific Computing
* Automation
* Web Development
* Game Development

And occasionally for creating bugs so confusing that future historians may study them.

---

## A Small Secret

Professional programmers don't choose Python because it's "easy."

They choose it because it helps them get work done.

Fast.

Readable code is easier to maintain.

And maintainable code is valuable.

---

## First Principles Takeaway

A computer is a very fast machine with absolutely no common sense.

Programming languages exist because humans need a way to communicate precise instructions to computers.

Python became popular because it makes that communication simpler, clearer, and more productive.

In short:

Computers are powerful.

Humans are creative.

Python helps the two cooperate.

Most of the time.
