# Python: From First Principles

> Understanding Python through logic, intuition, and first principles.

Welcome to **Python: From First Principles**.

Most Python tutorials focus on teaching syntax. This repository focuses on understanding the ideas behind the syntax.

The goal is not simply to learn how to write Python code, but to understand why programming concepts exist, what problems they solve, and how they fit together to create useful software.

Whether you're taking your first steps into programming or revisiting the fundamentals, this repository aims to build a strong foundation through reasoning, intuition, and hands-on projects.

---

## Why This Repository Exists

Programming is often taught as a collection of commands to memorize:

```python
for i in range(10):
    print(i)
```

But before learning *how* a loop works, it's worth asking:

* Why do loops exist?
* What problem are they solving?
* How would a human perform the same task?
* Why is repetition such an important idea in programming?

This repository approaches Python from that perspective.

Instead of treating programming as a language to memorize, we treat it as a way of thinking.

I'm here not only to learn, but also to share what I learn along the way.

---

## Learning Philosophy

A concept understood becomes a tool.

A rule memorized becomes a limitation.

Every lesson in this repository is designed around:

* Logic before syntax
* Understanding before memorization
* Problem-solving before implementation
* Curiosity before complexity
